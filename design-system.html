Wigo4it Designsysteem

Dit systeem vertaalt jullie bestaande pagina's naar herbruikbare designtokens en componenten op basis van TailwindCSS. Het behoudt de energieke persoonlijkheid van het merk met sociale impact, terwijl de implementatie sneller en consistenter wordt.

Merk-eerst Toegankelijk contrast Herbruikbare componenten Tailwind-vriendelijk

Tailwind config voorbeeld

tailwind.config = {
  theme: {
    extend: {
      colors: {
        brand: {
          darkGreen: '#005351',
          darkBlue: '#434d8e',
          aubergine: '#362c46',
          softYellow: '#e9eb86'
        }
      },
      fontFamily: {
        heading: ['PP Neue Machina'],
        body: ['Raleway']
      }
    }
  }
}

Fundamenten

Fundamenten bepalen het visuele DNA: kleur, typografie, spacing en elevatie. Componenten worden alleen opgebouwd vanuit deze bouwstenen.

Stem

Zelfverzekerd, praktisch en optimistisch. De interface voelt slim en benaderbaar.

Contrast

Gebruik diepe achtergronden met warme accenten. Primaire tekst blijft op donkergroen of aubergine.

Beweging

Kies voor reveals op sectieniveau en subtiele zwevende accenten in plaats van drukke micro-interacties.

Kleurtokens

Klik op een tokenkaart om de hex-waarde te kopieren.
